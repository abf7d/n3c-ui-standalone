Cumulative and Average COVID+ Cases

This dashboard allows for the tracking of COVID+ cases in the Enclave. Users can compare both daily and cumulative cases to the 7-day rolling average.

COVID+ Defined As:
  • Laboratory-confirmed positive COVID-19 PCR or Antigen test
  • (or) Laboratory-confirmed positive COVID-19 Antibody test
  • (or) Medical visit in which the ICD-10 code for COVID-19 (U07.1) was recorded
